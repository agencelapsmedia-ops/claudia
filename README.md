# CLAUDIA — Cybersecurity Intelligence Platform

**LAPSMEDIA x CyberTIA** | React Native + Expo

## Stack

- React Native + Expo SDK 51
- Expo Router (file-based navigation)
- TypeScript (strict)
- React Native Reanimated 3
- React Native SVG

## Getting Started

```bash
npm install
npx expo start
```

## Deploy to Vercel

```bash
npx expo export -p web
```

Or connect this repo to Vercel — it will auto-build using `vercel.json`.

## Project Structure

```
app/          → Screens (Expo Router file-based)
components/   → Reusable UI components
constants/    → Colors, typography, mock data
hooks/        → Custom React hooks
types/        → TypeScript interfaces
utils/        → Formatters & helpers
```
